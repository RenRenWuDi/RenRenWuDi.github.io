## 什么是收益聚合器？


收益聚合器（Yield Aggregator）自动帮用户找到DeFi中最高的收益，并将资金自动复利投资。

为什么需要收益聚合器？

• DeFi协议众多，收益率随时变化
• 手动切换需要频繁操作，Gas费高
• 收益聚合器自动帮用户"打组合拳"
• 小资金也能享受机构级收益策略


## Yearn Finance是什么？


Yearn是以太坊上最知名的收益聚合协议，由Andre Cronje创建。

Yearn的核心：

• 自动将资金分配到最高收益的协议
• 收益自动复利（Compound/Aave存入）
• 定期调整策略适应市场变化
• 用户只需存款，自动获得收益


## Vault（机枪池）机制


Yearn的Vault是自动策略执行器，用户存入资产，Yearn自动操作。

## APY vs APR


收益通常用APR或APY表示，两者区别很大。

APR（年化收益率）：

单利计算，不考虑复利
1000美元，年利率5%，一年后=1050美元

APY（有效年化收益率）：

复利计算
1000美元，月复利5%/12，12个月后=1051.16美元

uint256 public totalAssets;
mapping(address = uint256) public shares;
mapping(address = uint256) public lastReport;
uint256 public constant managementFee = 500;  // 2%管理费
function deposit(uint256 amount) external {
uint256 shares_ = amount;  // 简化：1:1
shares[msg.sender] += shares_;
totalAssets += amount;
IERC20(token).transferFrom(msg.sender, address(this), amount);
}
function withdraw(uint256 shares_) external {
uint256 amount = shares_;
shares[msg.sender] -= shares_;
totalAssets -= amount;
IERC20(token).transfer(msg.sender, amount);
}

## 收益聚合的风险


收益聚合的风险：

• 智能合约风险：Yearn代码可能被攻击
• 无常损失：如果策略包含AMM池
• 策略失败：自动切换到亏损策略
• 流动性风险：大量赎回时可能延迟
