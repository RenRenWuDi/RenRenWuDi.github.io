**结构化产品**


## 机构DeFi玩法大公开


## 为什么机构爱结构化产品？


传统金融里，结构化产品是把多种金融工具打包，满足不同风险偏好的投资者。DeFi里的结构化产品本质一样：通过收益分层，让保守型和激进型投资者各取所需。

机构入场DeFi，最喜欢用结构化产品——他们不可能拿10亿美元直接冲进Uniswap做LP。


## 收益分级（Yield Tranching）


收益分级把一个收益池分成两层：

优先层（Senior Tranche）：保守型投资者，先拿收益。收益稳定但上限低

劣后层（Junior Tranche）：激进型投资者，后拿收益。可能更高，也可能什么都拿不到

YieldTranche.sol


```solidity
// 简化版收益分级合约
contract YieldTranche {
uint256 public seniorDeposit;
uint256 public seniorRate = 5;    // 保底5%年化
uint256 public juniorDeposit;
uint256 public juniorMaxRate = 20;
```


```javascript
function distributeYield(uint256 totalYield) external {
uint256 seniorEntitlement = (seniorDeposit * seniorRate) / 100;
if(totalYield <= seniorEntitlement) {
distributeToSenior(totalYield); // 劣后层零收益
} else {
distributeToSenior(seniorEntitlement);
uint256 juniorShare = totalYield - seniorEntitlement;
uint256 juniorRate = juniorDeposit > 0
? (juniorShare * 100) / juniorDeposit : 0;
juniorRate = juniorRate > juniorMaxRate ? juniorMaxRate : juniorRate;
distributeToJunior(juniorShare);
}
}
function distributeToSenior(uint256) internal { /* ... */ }
function distributeToJunior(uint256) internal { /* ... */ }
}
```


类比：一锅粥10个人分。优先层先盛，保证能盛到粥；劣后层最后盛，如果粥不够可能什么都没有。但如果粥很多，最后的人反而喝到更多。


## 杠杆代币


杠杆代币实现了"有杠杆但不需要保证金管理"的体验。

3倍做多ETH：ETH涨1%，代币涨3%；ETH跌1%，代币跌3%

自动再平衡：每天收盘时自动调整仓位，维持固定杠杆倍数

没有清算风险：持有杠杆代币不会被强制清算，但会有磨损


## 利率结构化


机构在DeFi里最常用的工具是利率结构化——把不稳定的浮动利率，变成固定利率或区间利率。

固定利率：把手头的浮动收益换成"保本+固定利息"，适合保守机构资金

区间利率：在某个利率区间内收益，超过部分被自动"卖掉"


## 散户能参与结构化产品吗？
