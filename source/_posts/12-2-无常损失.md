**无常损失**


## 收益背后的代价


## 什么是无常损失？


无常损失（Impermanent Loss，IL）是DeFi LP的隐形杀手。它不是费用，而是当你提供流动性时，价格变化带来的账面损失。

通俗解释：

你持有1个ETH价格1000美元，放进LP池变成了0.5 ETH + 500 USDC。ETH涨到2000美元时，池子变成约0.35 ETH + 700 USDC，总价值1400美元。但持有不动是2000美元——差了600美元，这就是无常损失。


## 无常损失的计算


ILCalculator.sol


```solidity
// 无常损失计算
contract ILCalculator {
// IL公式: 2*sqrt(r) / (1+r) - 1
// r = 最终价格 / 初始价格
function impermanentLoss(uint256 priceRatio)
public pure returns(int256 ilBps)
{
uint256 r = priceRatio;
uint256 sqrtR = sqrt(r * 1e18);
uint256 lpValue = (2 * sqrtR * 1e18) / (1e18 + r);
return int256(1e18) - int256(lpValue);
}
```

```javascript
function sqrt(uint256 x) internal pure returns(uint256 y) {
y = x;
uint256 z = (x + 1) / 2;
while(z < y){ y = z; z = (x/z + z) / 2; }
}
}
```


记住这个规律：价格变化越大，无常损失越严重。50%涨跌约1.55%，100%涨跌约4.2%，200%涨跌约10.36%。


## 什么时候无常损失最可怕？


单边行情：ETH持续上涨或下跌，无常损失持续扩大，永远无法恢复

高波动币对：小币种的波动性会让无常损失迅速吃掉全部收益

稳定币对（USDC/USDT）：理论上无常损失为零，但年化收益也极低


## 机枪池：帮你自动逃离无常损失


机枪池（Yield Aggregator）的作用是：监控LP仓位，发现无常损失过大时，自动撤出换一个策略。

机枪池的核心逻辑：

* 监控目标池的年化收益和无常损失
* 当年化收益 > 无常损失 + 手续费时，继续持有
* 当无常损失超过收益时，自动撤出并换池
* Yearn Finance是最知名的机枪池，策略库里有超过50种策略


## 实际配置建议


低风险：50%稳定币 + 50%稳定币LP（Curve的USDC/USDT池），年化3-8%，几乎无无常损失

中风险：70%主流币LP + 30%稳定币（ETH/USDC或BTC/ETH池），年化10-20%

高风险：小币种高波动LP，年化30-100%，但无常损失可能吃掉全部收益
