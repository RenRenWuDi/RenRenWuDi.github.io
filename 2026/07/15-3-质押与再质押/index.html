原生质押、LST、EigenLayer——收益来源与风险对比

系列十五「撸毛与赚钱」第3篇

💡 质押ETH赚多少？LST和直接质押哪个好？再质押的风险在哪？

⏱ 阅读时间：约12分钟

1. 质押的本质：用安全换收益

质押（Staking）是PoS区块链的核心机制：你把Token锁到验证节点，节点验证交易获取区块奖励——你赚取这部分收益。你的Token换来的是网络安全。

质押收益的来源：\n\n🏗 区块奖励（通胀）：新铸造的Token分给质押者\n💰 交易费用：用户支付的交易手续费\n🛡 MEV收益：验证节点可以捕获MEV利润\n\n实际年化：ETH约3-4%（扣除通胀后约2-3%）、SOL约7-8%

2. 原生质押 vs LST

你有两种方式质押ETH：

原生质押（Solo Staking）：运行自己的验证节点（32 ETH门槛），赚取全部收益

LST（流动性质押Token）：通过Lido/Rocket Pool存入ETH，获取stETH/rETH——随时可以卖出

CEX质押：在币安/Coinbase质押，平台收服务费），手续费高但无门槛

contract.sol

LST的核心优势：流动性。原生验证节点里的ETH不能动，但stETH可以随时在二级市场卖出。你获得了ETH的收益率+流动性。

3. EigenLayer再质押：风险和收益都放大

再质押（Restaking）让你把已经质押的ETH（通过LST）再质押一次——支持更多协议的安全性，获取额外收益。

收益：AVS验证服务费 + EigenLayer代币激励

风险：削减（Slashing）——如果AVS验证行为违规，你的质押代币会被扣掉

超额抵押：每个AVS对你的stETH有不同的”削减率”

再质押的关键问题：\n\n📊 额外收益：再质押年产5-15% APY（含代币激励）\n⚠️ 额外风险：每个AVS都有独立的削减条件\n\n如果你存入100 ETH做EigenLayer再质押，\n但在5个AVS中被削减，每个削减5%\n→ 总损失高达25%的stETH

4. 做质押前要问自己的问题

  1. 锁定期：可以在需要资金时随时撤出吗？

  2. 削减风险：如果验证节点出问题，我的Token会损失多少？

  3. 操作成本：Gas费、CEX提币费、创建验证节点时间成本

  4. 税务影响：质押收益算收入还是资本利得？

质押决策矩阵：\n\n🟢 大额（>32 ETH）+长期持有 = Solo Staking\n🟢 小额+想要流动性 = Lido LST\n🟡 追求更高收益+接受风险 = EigenLayer再质押（高风险）\n🟢 最稳妥 = 不做任何质押，直接持有ETH

5. 下篇预告

下一期讲测试网参与：测试网空投的历史机会、节点搭建基础、如何高效参与测试网空投。

✅ 质押是用ETH安全性换取区块奖励+交易费+MEV收益

✅ 原生质押（>32 ETH）：收益最高但无流动性

✅ LST（Lido/rETH）：收益略低但有流动性和灵活性

✅ EigenLayer再质押：额外收益和额外削减风险并存

✅ 做质押前问自己：锁定期、削减风险、成本、税务
