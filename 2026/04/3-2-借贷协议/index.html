借贷协议
Compound的利率模型与抵押机制
系列三「DeFi原理与实战」第2篇
💡 你将学到：存款利率、贷款利率、流动性池、抵押机制、清算逻辑
⏱ 阅读时间：约15分钟

  1. 什么是借贷协议？
    借贷协议是DeFi的核心组成部分，允许用户存款赚利息，或抵押资产借款。
    借贷协议 vs 传统银行：

• 传统银行：中心化、需KYC、有地域限制
• 借贷协议：去中心化、无需许可、全球可用
• 利率由市场供需自动调节
• 抵押品自动清算
2. Compound的工作原理
Compound是以太坊上最流行的借贷协议之一。
核心机制：

• 用户存款获得cToken（代表存款凭证）
• 存款产生利息，cToken价值增长
• 用户抵押存款借出其他资产
• 借款需要超额抵押
• 利率由利用率决定
利率模型
Compound使用基于利用率的利率模型：
存款利率 = 基础利率 + 利用率 * 斜率
贷款利率 = 存款利率 * (1 + 储备金率)

利用率 = 已借金额 / 总存款金额
InterestModel.sol
function calculateInterest(uint256 utilization) internal pure returns (uint256) {
uint256 baseRate = 2e18 / 365 days;
uint256 slope1 = 0.05e18 / 365 days;
uint256 slope2 = 0.4e18 / 365 days;
if (utilization <= 0.5e18) {
return baseRate + utilization * 2 * slope1;
} else {
return baseRate + slope1 + (utilization - 0.5e18) * 2 * slope2;
}
}

  1. cToken机制
    cToken是Compound的存款凭证，持有cToken代表你在Compound中的存款份额。
  2. 抵押借款流程
    步骤1：存款（如存入1000 USDC，获得1000 cUSDC）
    步骤2：开启抵押（启用cUSDC作为抵押品）
    步骤3：借款（如借出500 USDC）
    步骤4：归还借款+利息
    步骤5：取回抵押品
    抵押率与清算
    清算阈值：

• 每个资产有不同的抵押因子（50%-90%）
• 清算发生在 抵押品价值/借款价值 清算阈值
• 清算人可以用折扣价格偿还债务并获得抵押品
• 保护协议不产生坏账
5. 完整借贷合约示例
SimpleLending.sol
contract SimpleLending {
mapping(address = uint256) public deposits;
mapping(address = uint256) public borrows;
uint256 public totalDeposits;
uint256 public totalBorrows;
function deposit() external payable {
deposits[msg.sender] += msg.value;
totalDeposits += msg.value;
}
function borrow(uint256 amount) external {
require(deposits[msg.sender] 0);
uint256 maxBorrow = deposits[msg.sender] * 75 / 100;
require(amount = maxBorrow);
borrows[msg.sender] += amount;
totalBorrows += amount;
payable(msg.sender).transfer(amount);
}
function repay() external payable {
require(borrows[msg.sender] 0);
require(msg.value = borrows[msg.sender]);
totalBorrows -= borrows[msg.sender];
borrows[msg.sender] = 0;
}
}

  1. 闪电贷（Flash Loan）
    闪电贷是一种无需抵押品的即时贷款，必须在同一交易内归还。
    闪电贷用途：

• 套利：低价买入、高价卖出
• 抵押品交换：用一种资产换另一种
• 债务重组：即时平仓并开新仓
• 风险：如果交易失败，贷款自动撤销
7. 总结

① 借贷协议让存款获得利息，抵押资产获得流动性
② 利率由市场供需自动调节，利用率越高利率越高
③ cToken代表存款份额，价值随利息增长
④ 超额抵押+清算机制保护协议安全
⑤ 闪电贷无需抵押品，但必须在同一交易内归还
⑥ Compound是借贷协议的行业标杆
